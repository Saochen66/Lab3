# Lab1 — 黑盒测试（queryBridgeWords）

## 5.1 所选被测函数及其需求规约

- 被测函数：`String queryBridgeWords(String word1, String word2)` — 查询桥接词

- 函数功能：
  给定已构建的有向词图 G 和两个单词 `word1`, `word2`，查找所有桥接词（bridge words）。桥接词定义为在图中同时存在边 `word1 -> b` 和 `b -> word2` 的节点 `b`。函数返回所有桥接词的字符串列表（以逗号和空格分隔），若不存在桥接词则返回说明性信息；若输入无效则返回错误提示字符串。

- 输入描述：
  - `word1`：非空字符串，表示起始单词（允许字母大小写，但按实现约定应做大小写归一化）。
  - `word2`：非空字符串，表示目标单词。
  - 假定图 `G` 在函数外已正确加载并可访问。

- 输出描述：
  - 当存在桥接词时，返回形式为 `b1, b2, ...`（单词按任意确定顺序列出，逗号+空格分隔）。
  - 当不存在桥接词时，返回 `No bridge words from '<word1>' to '<word2>'.` 或等价的说明字符串。
  - 当输入单词不存在或输入不合法（如空字符串、null、包含非法字符）时，返回相应的错误提示，例如 `Word '<word>' not found in the graph.` 或 `Invalid input: <说明>`。

- 额外假定：函数对大小写不敏感（会在内部归一化），不会抛出未处理的异常而是返回错误提示字符串；输入两端应做 trim 处理。
